Play this Breakout-inspired game using by opening the HTML file (ensuring the CSS and JavaScript files are in the same folder)! Use your paddle to bounce the ball and break the bricks to increase your score. Use the left/right arrow keys or A/D to move the paddle, and press V to show the variables (ball direction, x and y velocity, and the number of blocks on screen). Press [space] to reset the game.
